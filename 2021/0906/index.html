Windowsの gitの改行回りは, .gitattributes が必須なのかと思っていたけど下記だけで良かった. git bashインストール時の設定で勝手に設定されるものだけど, 以下にしておけば Windowsはとりあえず無難. エディタや IDEの改行設定を返る必要も特にない

 git config --global core.autocrlf true
